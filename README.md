# Gaming Tracker

Progress: 0%
Priority: High
Status: In progress

### About project

This project is a gaming tracker that is dedicated to helping me keep count of the number of games I have played

and also wish to play at some point in the future,

# Phases

- **Phase 1: Scope**
    - **Data**: Game entry need Title, Platform(PC, etc), Status(In progress, Wishlist)
    - **Features**:
        - Add new game.
        - View list of games(Completed/Wishlist)
        - Update game statues(Wishlist to Completed)
        - Delete game
- **Phase 2: Design**
    - Blueprint(Game.java):
        - Variable to consider
            - String title
            - String platform
            - String status
    - Concrete calls/Manager(GameTracker.java):
        - Focuses
            - Collections of games(Arryas)
            - Functions/logics
                - adding games
                - removing games
                - wishlist
                - completed games
    - Frontend:
        - Interactive system
- **Phase 3: Tools/Implementaions**
    - VSCODE
    - Git(for version control)
    - JavaFX/Swing(Interface)
    - Spring-Boot(database)
